# EFX CLI

Command Line Interface for the EFX Toolkit.

## Requirements

The EFX CLI requires Java 11 or later.

## Building

You can build this project using Maven:

```bash
mvn clean package
```

This will create an executable jar in the `target` directory.

## Running

```bash
java -jar target/efx-cli-1.0.0-SNAPSHOT.jar --help
```
